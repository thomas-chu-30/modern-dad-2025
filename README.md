# website 錄音功能

## 技術棧

- Node.js
- Express.js
- EJS (模板引擎)
- Nodemon (開發環境自動重啟)

## 安裝說明

1. 確保已安裝 Node.js
2. 克隆專案

```bash
git clone https://github.com/thomas-chu-30/modern-dad-2025.git
```

3. 安裝依賴

```bash
npm install
```

4. 啟動專案

```bash
npm start
```

## 專案結構

```
modern-dad-2025/
├── bin/            # 應用程式啟動腳本
├── public/         # 靜態資源文件
├── routes/         # 路由處理
├── views/          # EJS 模板文件
├── app.js          # 應用程式主文件
└── package.json    # 專案配置和依賴
```
